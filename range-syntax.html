Range Syntax

Can I Use: 90%

The CSS media query range syntax improves legibility of media queries by making use of comparison operators: >, <, >=, or <=

Old


  @media (min-width: 600px) {}
  @media (min-width: 600px) and (max-width: 800px) {}

New


  @media (width >= 600px) {}
  @media (600px <= width <= 800px) {}

Example


.intro { color:black; }
@media (800px <= width <= 1200px) {
  .intro { color: blue; }
}
@media (width >= 1200px) {
  .intro { color: orange; }
}
