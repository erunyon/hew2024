Cascade Layers

Can I Use: 94%

CSS Cascade Layers provide a way to organize CSS and provide an easier way to override earlier styles.

Demo 1: Standard Cascade

<p class="indigo"><strong>INIGO:</strong> Hello there.</p>

<p><strong>MAN IN BLACK:</strong> …</p>

<p class="indigo"><strong>INIGO:</strong> Sorry.</p>

<p><strong>MAN IN BLACK:</strong> Thank you.</p> 

p { color:#333; }
.indigo { color:indigo; }
p { color:orange; }

INIGO: Hello there.

MAN IN BLACK: Look, I don't mean to be rude, but this is not as easy as it looks. So I'd appreciate it if you wouldn't distract me.

INIGO: Sorry.

MAN IN BLACK: Thank you.
