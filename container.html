@container

Can I Use: 91%

The @container CSS at-rule is a conditional group rule that applies styles to a containment context.


<div class="parent">
  <div class="child">
    <h2>Child title</h2>
    <p>Child content</p>
  </div>
</div>


.parent {
  container-type: inline-size;
}
@container (min-width: 700px) {
  .child {
    /* styles */
  }
}
