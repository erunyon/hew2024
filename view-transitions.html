@view-transition (Chrome only)

The view-transition() CSS at-rule is used to opt in the current and destination documents to undergo a view transition, in the case of a cross-document navigation.


@view-transition { navigation:auto; }

@media not (prefers-reduced-motion: reduce) {
  h1 { view-transition-name: article-heading; }

  .news-listing-1234 { view-transition-name: news-title }
  .news-detail-1234 { view-transition-name: news-title }
}
