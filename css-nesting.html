CSS Nesting

Can I Use: 87%

The CSS nesting module allows you to write your stylesheets so that they are easier to read, more modular, and more maintainable. As you are not constantly repeating selectors, the file size can also be reduced.


.card { background: gray;
  a { text-decoration: none; }
  .card-title { font-size: 1.5rem; }
  .card-body { font-size: 1rem; }
}

/* is equivalent to */

.card { background: gray; }
.card a { text-decoration: none; }
.card .card-title { font-size: 1.5rem; }
.card .card-body { font-size: 1rem; }

Nesting @-rules

  • @media
  • @supports
  • @layer
  • @scope
  • @container

Example


.card-grid {
  display: grid;
  @media (width >= 1200px) {
    & {
      grid-auto-flow: column;
    }
  }
}

/* is equivalent to */

.card-grid { display: grid; }
@media (width >= 1200px) {
  .card-grid { grid-auto-flow: column; }
}
