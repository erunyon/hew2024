@property

Can I Use: 90%

The @property CSS at-rule allows developers to explicitly define CSS custom properties, allowing for property type checking and constraining, setting default values, and defining whether a custom property can inherit values or not.


@property --property-name {
  syntax: "<percentage>";
  inherits: false; /* default is true */
  initial-value: 10%;
}

Available `syntax` values

  • <length> - values with units (10px, 10vw)
  • <number> - any numeric value
  • <integer> - whole numbers
  • <percentage> - must include the `%` (13%)
  • <length-percentage> - accepts valid <length> or <percentage> values
  • <color> - any valid color such as hex or rgb
  • <image>
  • <url>
  • <angle>
  • <time>
  • <resolution>
  • <transform-function>
  • <custom-ident>

Demo


@property --my-color {
  syntax: "<color>";
  inherits: false;
  initial-value: #00f;
}
ul {
  --my-color: orange;
}
li {
  color: white;
  background-color: var(--my-color);
}
