dialog

Can I Use: 95%

The <dialog> HTML element represents a modal or non-modal dialog box or other interactive component, such as a dismissible alert, inspector, or subwindow..


<dialog id="dialog-1">
  <h2>This is my sweet modal</h2>
  <p>More info here.</p>
  <form method="dialog">
    <button>Close dialog</button>
  </form>
</dialog>

<button onClick="document.getElementById('dialog-1').showModal();" type="button">Open Dialog</button>

dialog {
  opacity: 0;
  transition:
    display 1s allow-discrete,
    overlay 1s allow-discrete,
    opacity 1s;

  &[open] {
    opacity: 1;
    @starting-style {
      opacity: 0;
    }
  }
