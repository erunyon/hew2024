text-wrap

Balance

Can I Use: 84%

Text is wrapped in a way that best balances the number of characters on each line. Supports only a limited number of lines (six or less for Chromium and ten or less for Firefox).

Pretty (Chrome only)

Can I Use: 70%

Results in the same behavior as wrap, except it uses a slower algorithm that favors better layout over speed. This is intended for body copy where good typography is favored over performance.


.balance {
  text-wrap: balance;
}

.pretty {
  text-wrap: pretty;
}
