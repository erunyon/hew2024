:user-valid/:user-invalid

Can I Use: 87%

The :user-valid/user-invalid pseudo-classes represents any validated form element whose value validates correctly based on its validation constraints. However, unlike :valid it only matches once the user has interacted with it.


input:user-valid {
  border:2px solid green;
}
input:user-invalid {
  border:2px solid red;
}

label:has(+ input:user-valid)::after
label:has(+ input:user-invalid)::after {
  display:inline-block; margin-left:0.4rem;
}
label:has(+ input:user-valid)::after {
  content:"✓";
  color:green;
}
label:has(+ input:user-invalid)::after {
  content:"✖";
  color:red;
}
