:empty

Can I Use: 96%

The :empty pseudo-class represents any element that has no children. Only element nodes and text (including whitespace) are considered. Comments do not affect whether an element is considered empty or not.

This div is considered empty:

<div></div>

While this div is not:

<div> </div>

:empty { background-color:lightblue; }
<ul>

</ul>
