@hover

Can I Use: 96%

The @hover CSS media feature can be used to test whether the user's primary input mechanism can hover over elements.


.card-body {
  position:absolute;
  inset:auto 0 0;
}
a:hover {
  .card-body { inset:auto 0 0; }
}
@media (hover: hover) {
  .card-body { inset:auto 0 -1.8em; }
}
