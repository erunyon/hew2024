:any-link

Can I Use: 96%

The :any-link CSS pseudo-class selector represents an element that acts as the source anchor of a hyperlink, independent of whether it has been visited. In other words, it matches every <a> that has an href attribute. 


a, a:link, a:visited { color:red; }
:any-link { background-color:lightblue; }
